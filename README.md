# Johncuba Tap

## How do I install these formulae?

`brew install johncuba/tap/<formula>`

Or `brew tap johncuba/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "johncuba/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
